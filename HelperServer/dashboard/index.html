First run guide

1
Start DEX_Helper.exe. The top-left status should say active and Script delivery should say ready.
2
Copy the loadstring from the sidebar and run it in Roblox. The Roblox session card should switch from waiting to connected.
3
Open Search Center in DEX. Keep Low ON for smooth sessions, or use Full only when you can tolerate stutter.
4
Press Index. This dashboard will show the exact game, progress, cached scripts, failed scripts, and helper index size.
5
Search here after the index has source. This keeps browsing, ranking, and analysis outside the Roblox UI.
